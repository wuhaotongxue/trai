## 1. 中文标点禁令

<div style="background:#FFF4F4;border:1px solid #FFB4B4;border-radius:8px;padding:12px 16px;margin:12px 0;">
  <strong style="color:#D32F2F;">&#x274C; 绝对禁止</strong> — 代码、注释中严禁出现中文全角标点
  <div style="margin-top:8px;font-size:13px;">
    <span style="color:#D32F2F;">&#x2718;</span> <code style="color:#D32F2F;">，。！？：</code>
    &nbsp;&nbsp;
    <span style="color:#2E7D32;">&#x2714;</span> <code style="color:#2E7D32;">, . ! ? :</code>
  </div>
</div>

## 2. 核心原则

<div style="background:#FFF9C4;border:1px solid #FFF176;border-radius:8px;padding:12px 16px;margin:12px 0;">
  <strong style="color:#F57F17;">&#x26A0; 核心铁律</strong>
  <ul style="margin:8px 0 0 0;padding-left:20px;font-size:13px;">
    <li><strong>所有用户可见文案必须通过 <code>useTranslations</code> 获取</strong></li>
    <li><strong>严禁在 JSX/TSX 文件中直接写中文字符串</strong></li>
  </ul>
</div>

## 3. 翻译文件路径

```
src/messages/
├── zh.json    # 中文
├── en.json    # 英文
├── ja.json    # 日文
└── ko.json    # 韩文
```

## 4. Key 命名规范

<div style="display:grid;grid-template-columns:1fr 1fr;gap:12px;margin:12px 0;">
  <div style="background:#E8F5E9;border:1px solid #A5D6A7;border-radius:8px;padding:12px;">
    <strong style="color:#2E7D32;">&#x2714; 正确格式</strong>
    <ul style="margin:8px 0 0 0;padding-left:16px;font-size:13px;color:#555;">
      <li>层级结构，点分隔</li>
      <li>全部小写</li>
      <li><code>"login.form.email.label"</code></li>
      <li><code>"common.message.success"</code></li>
    </ul>
  </div>
  <div style="background:#FFEBEE;border:1px solid #FFCDD2;border-radius:8px;padding:12px;">
    <strong style="color:#C62828;">&#x274C; 禁止格式</strong>
    <ul style="margin:8px 0 0 0;padding-left:16px;font-size:13px;color:#555;">
      <li>驼峰命名</li>
      <li>大写字母</li>
      <li>横杠连接</li>
      <li><code>"loginFormEmailLabel"</code></li>
    </ul>
  </div>
</div>

## 5. 使用方式

| 类型 | 规则 | 示例 |
|------|------|------|
| 锁定命名空间 (推荐) | `useTranslations("login.form")` | `t("email.label")` |
| 根命名空间 | `useTranslations()` | `t("common.pageTitle")` |
| 带参数 | `t("greeting", { username: "..." })` | `{"greeting": "Welcome, {username}"}` |

## 7. 常见错误

<div style="display:grid;grid-template-columns:1fr 1fr;gap:12px;margin:12px 0;">

  <div style="background:#FFEBEE;border:1px solid #FFCDD2;border-radius:8px;padding:12px;">
    <strong style="color:#C62828;font-size:13px;">&#x274C; JSX 中直接写中文</strong>
    <div style="font-size:12px;color:#C62828;margin-top:8px;">
      必须用 <code>t("meeting.create")</code>
    </div>
  </div>

  <div style="background:#FFEBEE;border:1px solid #FFCDD2;border-radius:8px;padding:12px;">
    <strong style="color:#C62828;font-size:13px;">&#x274C; Key 与结构不匹配</strong>
    <div style="font-size:12px;color:#C62828;margin-top:8px;">
      锁定后用 <code>t("form.email")</code>
    </div>
  </div>

  <div style="background:#FFEBEE;border:1px solid #FFCDD2;border-radius:8px;padding:12px;">
    <strong style="color:#C62828;font-size:13px;">&#x274C; 只填中文</strong>
    <div style="font-size:12px;color:#C62828;margin-top:8px;">
      四种语言必须同步添加
    </div>
  </div>

  <div style="background:#FFEBEE;border:1px solid #FFCDD2;border-radius:8px;padding:12px;">
    <strong style="color:#C62828;font-size:13px;">&#x274C; Key 大写或横杠</strong>
    <div style="font-size:12px;color:#C62828;margin-top:8px;">
      全小写，点分隔
    </div>
  </div>

</div>

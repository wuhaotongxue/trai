# Frontend_Next_js_Admin_后台管理规范

---

## 1. 路由结构

```
src/app/[locale]/admin/
├── page.tsx              # Admin Dashboard
├── layout.tsx            # Admin Layout
├── users/                # 用户管理
│   └── page.tsx
└── system/               # 系统设置
    └── page.tsx
```

---

## 2. 布局规范

### 2.1 Admin Layout

| 区域 | 组件 | 说明 |
|------|------|------|
| 左侧 | `AdminSidebar` | 管理员导航 |
| 右侧 | `children` | 页面内容 |

**权限检查**

| 检查项 | 失败处理 |
|--------|---------|
| 是否登录 | 跳转 `/login` |
| 是否 Admin | 跳转 `/dashboard` |

**实现参考**：`frontend_next/src/app/[locale]/admin/layout.tsx`

### 2.2 Admin Sidebar

| 菜单项 | 路径 | 说明 |
|--------|------|------|
| Dashboard | `/admin` | 管理概览 |
| 用户管理 | `/admin/users` | 用户列表/编辑 |
| 系统设置 | `/admin/system` | 系统配置 |

---

## 3. 用户管理页面

### 3.1 功能模块

| 功能 | 说明 |
|------|------|
| 用户列表 | 分页展示所有用户 |
| 用户搜索 | 按用户名/邮箱搜索 |
| 角色管理 | 修改用户角色 |
| 配额管理 | 修改用户配额 |
| 禁用/启用 | 账户状态管理 |

### 3.2 数据表格

| 列 | 说明 |
|------|------|
| 用户名 | 显示昵称 |
| 邮箱 | 登录邮箱 |
| 角色 | guest/user/vip/admin |
| 配额 | 剩余用量 |
| 状态 | 正常/禁用 |
| 操作 | 编辑/禁用 |

---

## 4. 系统设置页面

### 4.1 设置分组

| 分组 | 内容 |
|------|------|
| AI 模型 | 模型列表/价格 |
| 权限配置 | 角色权限 |
| 系统参数 | 配额限制 |
| 日志审计 | 操作日志 |

### 4.2 表单项

| 类型 | 组件 |
|------|------|
| 文本输入 | `Input` |
| 数字输入 | `Input type="number"` |
| 开关 | `Switch` |
| 下拉选择 | `Select` |
| 提交按钮 | `Button type="submit"` |

---

## 5. 禁止事项

- 非 admin 访问管理页面
- 修改权限无确认
- 批量操作无二次确认
- 敏感操作无日志
- 修改后无成功提示

# Frontend - Admin 用户管理规范

---

## 1. 用户列表

| 列 | 说明 |
|------|------|
| 用户名 | 用户昵称 |
| 邮箱 | 登录邮箱 |
| 角色 | guest/user/vip/admin |
| 操作 | 编辑/删除 |

---

## 2. 角色管理

| 角色 | 权限 |
|------|------|
| guest | 访客，极低配额 |
| user | 普通用户，基础配额 |
| vip | VIP 用户，无限配额 |
| admin | 管理员，全部权限 |

---

## 3. 禁止事项

- 管理员降级自己
- 删除最后一个 admin
- 批量删除无确认
- 角色变更无审计日志

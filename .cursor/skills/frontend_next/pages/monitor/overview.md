# Frontend_Next_js_Monitor_大屏展示规范

---

## 1. 路由位置

```
src/app/[locale]/dashboard/monitor/page.tsx
```

---

## 2. 布局结构

### 2.1 整体布局

| 区域 | 位置 | 组件 |
|------|------|------|
| 全屏控制 | 右上角 | `FullscreenButton` |
| 顶部标题栏 | header | 标题 + 时间 |
| 主内容 | main | 三栏布局 |
| 底部图表 | footer | 趋势图 |

### 2.2 布局比例

| 区域 | Tailwind 类 | 说明 |
|------|-------------|------|
| 左侧面板 | `col-span-3` | 统计数据 |
| 中间面板 | `col-span-6` | 核心图表 |
| 右侧面板 | `col-span-3` | 实时数据 |

### 2.3 样式规范

| 规范 | 说明 |
|------|------|
| 背景色 | `bg-slate-950` |
| 文字色 | `text-white` |
| 边框色 | `border-slate-800` |
| 溢出 | `overflow-hidden` |

**实现参考**：`frontend_next/src/app/[locale]/dashboard/monitor/page.tsx`

## 3. 三栏内容

### 3.1 左侧面板

| 内容 | 说明 |
|------|------|
| 用户统计 | 今日/本周/本月 |
| AI 使用统计 | 生成次数 |
| 会议统计 | 进行中/已完成 |

### 3.2 中间面板

| 内容 | 说明 |
|------|------|
| 实时数据 | 滚动显示 |
| 核心图表 | 折线/柱状图 |

### 3.3 右侧面板

| 内容 | 说明 |
|------|------|
| 实时告警 | 最新告警列表 |
| 系统状态 | 服务健康状态 |

## 4. 禁止事项

- 页面不自动刷新
- 图表数据不实时更新
- 缺少全屏功能
- 移动端未适配
- 颜色对比度不足
